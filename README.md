# 皮皮猪状态面板 🦞

一个专为移动设备设计的皮皮猪（个人管家）实时状态展示面板，使用用户提供的图片作为背景。

## 功能特性

- 🦞 **实时状态显示** - 助手名称、当前模型、在线状态
- ⏰ **运行信息** - 运行时间、任务队列、API调用统计
- 🛠️ **技能展示** - 可用技能列表（QVeris、Apple Notes等）
- 🔄 **自动刷新** - 每30秒自动更新状态
- 📱 **移动优先** - 专为手机屏幕优化设计
- 🎨 **现代化UI** - 白色大圆角卡片叠加在背景图片上
- 💫 **动画效果** - 平滑过渡和加载动画
- 🌙 **暗色模式** - 自动跟随系统主题
- 📱 **PWA支持** - 可安装到手机主屏幕

## 设计说明

### 布局结构
1. **背景**：用户提供的旅行图片作为全屏背景
2. **卡片**：白色大圆角卡片叠加在图片中下方位置
3. **内容**：卡片内显示皮皮猪的实时状态信息

### 状态信息
- **助手名称**：皮皮猪 🐽
- **当前模型**：deepseek/deepseek-chat（可切换）
- **在线状态**：运行中（实时显示）
- **运行时间**：X小时X分钟（自动更新）
- **任务队列**：X个待处理（颜色提示）
- **API调用**：今日X次（颜色提示）
- **可用技能**：15个已加载（标签展示）
- **运行环境**：macOS · OpenClaw
- **创建时间**：2026-02-25
- **最后更新**：刚刚/X分钟前（自动更新）

## 使用方法

### 在线访问
1. 开启 GitHub Pages 后访问部署链接
2. 或在本地服务器运行

### 本地运行
```bash
# 使用 Python 简单服务器
python3 -m http.server 8000

# 或使用 Node.js
npx serve .
```

然后在浏览器中打开：`http://localhost:8000`

### 安装到手机
1. 在手机浏览器中打开应用
2. 点击分享按钮 → "添加到主屏幕"
3. 应用将像原生应用一样运行

## 交互功能

### 按钮操作
- **刷新状态**：手动更新所有状态信息
- **切换模型**：打开模型选择模态框
- **查看文档**：打开 OpenClaw 文档（新标签页）

### 键盘快捷键
- **R**：刷新状态
- **ESC**：关闭模态框

### 自动功能
- 每30秒自动刷新状态
- 页面隐藏时暂停刷新，显示时恢复
- 网络状态检测（在线/离线提示）
